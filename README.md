## Apache Kafka Beginners Course Project

### About

A beginners course project to learn Apache Kafka basics.

Uses Java 11 from Amazon (**[Amazon Correto 11](https://docs.aws.amazon.com/corretto/latest/corretto-11-ug/windows-7-install.html)** on Windows and WSL2 (Debian 12)).

The project is set up to run in Docker Container (for UI interface: **[conduktor.io](https://www.conduktor.io/get-started/)**)

For local instance, the project runs on WSL2 by starting KRaft first.
